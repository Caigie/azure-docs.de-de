# <a name="azure-synapse-analytics-data-management-partners"></a>Partner für die Azure Synapse Analytics-Datenverwaltung

Für das Erstellen einer Data Warehouse-Lösung mit dem Feature „Synapse SQL-Pool“ von Azure Synapse steht eine große Auswahl branchenführender Tools zur Auswahl. In diesem Artikel sind Partnerunternehmen von Microsoft aufgeführt, die Tools und Lösungen für die Datenverwaltung mit Unterstützung von Azure Synapse anbieten.
