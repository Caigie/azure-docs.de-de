# <a name="azure-synapse-analytics-business-intelligence-partners"></a>Business Intelligence Partner für Azure Synapse Analytics

Wenn Sie Ihre gesamte Data Warehouse-Lösung mit einem Synapse SQL-Pool in Azure Synapse erstellen möchten, können Sie aus einer Vielzahl von branchenführenden Tools auswählen.  In diesem Artikel sind Partnerunternehmen von Microsoft aufgeführt, die offizielle Business Intelligence-Lösungen (BI) für die Datenverwaltung anbieten, die Azure SQL Data Warehouse unterstützen.
