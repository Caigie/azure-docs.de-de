# <a name="overview-of-the-nist-sp-800-53-r4-blueprint-sample"></a>Übersicht über das Blaupausenbeispiel „NIST SP 800-53 R4“

Das Blaupausenbeispiel „NIST SP 800-53 R4“ bietet mithilfe von [Azure Policy](../../../policy/overview.md) Schutzmaßnahmen für Governance, die Ihnen beim Bewerten bestimmter NIST SP 800-53 R4-Steuerungen helfen. Diese Blaupause hilft Kunden bei der Bereitstellung eines Kernsatzes von Richtlinien für jede in Azure bereitgestellte Architektur, die NIST SP 800-53 R4-Steuerungen implementieren muss.

## <a name="control-mapping"></a>Steuerungszuordnung

Der Bereich für die Steuerungszuordnung enthält Details zu in dieser Blaupause enthaltenen Richtlinien. Außerdem wird erläutert, wie diese Richtlinien verschiedene Steuerungen in NIST SP 800-53 R4 behandeln. Bei der Zuweisung zu einer Architektur werden Ressourcen von Azure Policy auf Nichtkonformität mit den zugewiesenen Richtlinien überprüft. Weitere Informationen finden Sie unter [Was ist Azure Policy?](../../../policy/overview.md).
